# Projeto História do Corinthians (HTML + CSS)

Este projeto é um site desenvolvido em **HTML5 e CSS3** dedicado à história do **Sport Club Corinthians Paulista**, apresentando uma linha do tempo com seus principais acontecimentos, conquistas, ídolos e momentos marcantes da trajetória do clube.

O objetivo do projeto é demonstrar conhecimento em HTML, CSS, organização de conteúdo e criação de layouts responsivos, servindo tanto como estudo quanto como material para portfólio.

## Funcionalidades

- Estrutura completa utilizando **HTML semântico**
- Layout organizado com **CSS puro**
- Navegação lateral com links para cada seção da linha do tempo
- Responsividade implementada para telas menores (media queries)
- Seções com imagens, divisões e destaques estilizados
- Código limpo, indentado e fácil de entender
- Conteúdo completo sobre a história e títulos do Corinthians

## Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:

- Organização de seções em HTML
- Uso correto de `<nav>`, `<header>`, `<main>`, `<section>` e `<aside>`
- Indentação e limpeza de código
- Estilização avançada com CSS
- Responsividade com `@media`
- Navegação por âncoras
- Estruturação de um site completo sem frameworks
